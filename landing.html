The Augmented Reality Edge Network Architecture (ARENA) is a multi-user and multi-application environment to simplify the development of mixed reality.

connect

A directory service connects users to nearby servers and content. An application/service runtime supervisor that can dispatch programs across any network connected device.

customize

Define custom scenes, with 3D content and programs. The ARENA includes fine-grained access control and the ability to sandbox and isolate content and programs.

collaborate

Users in the physical space can see the same content anchored to the physical world in AR. Properties of all objects are networked to provide a consistent real-time view from any device or platform.

Collaborative Mixed Reality

ARENA simplifies many common development tasks for multiuser cross-platform mixed reality programming, including seamless support for a spectrum of experiences (VR, AR, XR), on multiple platforms, including tablets, phones, headsets, and desktop browsers. ARENA supports implicitly networked content, offering a multiuser experience in real-time and integrated accurate relocalization. It also allows to create and host user programs on any network-connected device with fine-grained access.
